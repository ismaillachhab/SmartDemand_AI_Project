# 📦 IA Demand Predictor : Optimisation de la Supply Chain

Ce projet est un **Système Intelligent** conçu pour estimer la demande future de produits logistiques. Il combine l'analyse de données (Data Science) et le déploiement d'une interface interactive et moderne via Streamlit.

## 📊 Aperçu du Projet
L'objectif est de transformer des données historiques de ventes en décisions stratégiques pour la gestion des stocks. Le système traite des données brutes de demande, les nettoie et utilise des modèles de Machine Learning pour prédire les unités nécessaires par produit et par entrepôt.

### Fonctionnalités clés :
- **Prétraitement de données massives** : Nettoyage automatique des erreurs de tokenisation et des formats de nombres (parenthèses, virgules).
- **Double Modélisation** : Comparaison en temps réel entre le **Gradient Boosting** et le **Random Forest**.
- **Interface Premium** : Design Dark Mode épuré avec sélection de la période (2024-2028).
